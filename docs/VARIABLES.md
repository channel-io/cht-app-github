# Environments
> Here you should write what are all of the configurations a user can enter when
using the project.

## Base
### Stage
- ENV: `STAGE`
- Type: `String`  
- Default: `'development'`

## API
### PUBLIC HTTP PORT
- ENV: `API_PUBLIC_HTTP_PORT`
- Type: `String`  
- Default: `'4000'`

## LOG
### LEVEL
- ENV: `LOG_LEVEL`
- Type: `String`
- Default: `'INFO'`
